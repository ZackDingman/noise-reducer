GitHub Pages — Zack's Audio Outpost Noise Reduction Tool
------------------------------------------------------

1. Create a public GitHub repository (e.g., noise-reducer).
2. Upload all these files into the repository root.
3. Go to Settings → Pages → Source: `main` branch, `/ (root)` folder → Save.
4. Your live tool will be at https://YOUR-USERNAME.github.io/noise-reducer/

To embed into Hostinger Website Builder, use an HTML/Embed block with:
<iframe src="https://YOUR-USERNAME.github.io/noise-reducer/" width="100%" height="800" style="border:none;"></iframe>
